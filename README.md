# decaf377-go

Go decaf377 primitives and verifier compatibility tests.

This local module is staged as the future `github.com/mizufinance/decaf377-go`
repository. The first concrete compatibility target is the current Orbis
SHA-256 FROST wire format. Penumbra `decaf377-rdsa` remains a separate API
surface.

## Packages

- `decaf377`: native point encoding/decoding, quotient equivalence, scalar
  handling, point addition, and scalar multiplication.
- `gnark`: reusable decaf377 gnark gadgets and native helpers for circuit
  tests.
- `orbisfrost`: verifier for the current Orbis SHA-256 FROST `(R, z)` wire
  format.
- `rdsa`: verifier for Penumbra-compatible `decaf377-rdsa` SpendAuth
  signatures.

## Verification

Run:

```sh
go test ./...
```

The Orbis interop test consumes `testdata/orbis_vectors.json`, generated by the
neighboring `orbis-rs` checkout:

```sh
cd ../penumbra/tmp/orbis-rs
ORBIS_VECTOR_OUT=../../../decaf377-go/testdata/orbis_vectors.json \
  cargo test --no-default-features --features decaf377 -p crypto \
  generate_orbis_frost_interop_vectors -- --nocapture
```
